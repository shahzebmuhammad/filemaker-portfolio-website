## ğŸ¨ Customization

### Update Contact Information
Edit the email link in the footer section of `index.html`:
```html
<a href="mailto:your-email@example.com">Email Me</a>
```

### Change Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1e40af;
    --secondary-color: #3b82f6;
    /* Add your custom colors */
}
```

### Add Your Projects
Replace the project images and content in the Projects section of `index.html`.
